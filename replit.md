# hackathon-ifsp-jcr-2026

## Project Overview
Repository created for the hackathon at IFSP Jacareí (Instituto Federal de São Paulo - Jacareí campus) during the 2026 Cultural Week.

The project uses Flutter/Dart for cross-platform development (mobile, web, desktop).

## Status
This repository is in its initial state — no application code has been added yet. The `.gitignore` is configured for Flutter/Dart projects.
